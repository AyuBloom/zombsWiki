# Disappearing Resources

## Phenomenon

Occasionally, Trees and Stones (and likely also Neutral Camps, but no such cases have been reported for them) previously found can disappear after some time in a server.

## Cause

Unknown.

## Examples

### April 2026

On 26th April, a discord user named Ander sat on a missing rock's place. The rock with the UID 657 is completely gone. 

<ImageWithDescription image="/asset/gameplay/bugs/active/rss_go_poof/rock.png" description="Ander sitting on the missing rock." />

A full server scan showed that a tree and a rock is completely gone. A normal map contains 400 trees, 400 rocks, and 25 neutral camps, but here it wasn't the case.
