# Scripting 

Scripting has been a core part withing the zombs.io community ever since the absence of new game updates. Over the years, many new exploits have been discovered by the community, giving way to new script functions that, for many, have fundamentally changed how the game is played.

The [Fundamentals](/gameplay/scripts/fundamentals/overview) category provides an overview of the fundamentals of scripting in zombs.io, from best practices to cautions.

Many scripts have been written by the community and have been published publicly or kept privately. The [Popular Scripts](/gameplay/scripts/popular_scripts/overview) category lists the most popular scripts that was either intentionally published or was leaked without consent from the developers of them.
